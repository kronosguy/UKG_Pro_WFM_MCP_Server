# Repository Catalog

## Purpose

This catalog defines the public repository ecosystem for the UKG WFM Academy. Each repository is scoped as a standalone academy course, enterprise implementation playbook, portfolio project, or technical enablement system.

Repository names follow:

```text
ukg-wfm-[topic-name]
```

Each repository includes course metadata, architecture guidance, modules, lessons, labs, assessments, governance controls, troubleshooting guidance, and a capstone simulation.

## Repository Metadata Standard

Every repository must define:

- Course Title
- Repository Name
- Difficulty
- Estimated Duration
- Target Audience
- Business Value
- Technical Value
- Operational Value
- Portfolio Deliverables
- Capstone Outcome
