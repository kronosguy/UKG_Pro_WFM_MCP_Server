## Scoring Scale

| Score | Description |
| --- | --- |
| 4 | production-ready, clear, complete, and enterprise-aligned |
| 3 | complete and usable with minor gaps |
| 2 | partially complete with material gaps |
| 1 | incomplete or not operationally usable |

## Evaluation Areas

### Business Alignment

- solves the stated operational problem
- accounts for healthcare workforce scale
- connects to payroll, compliance, staffing, or executive value

### Technical Quality

- uses clear architecture
- defines inputs and outputs
- includes validation
- avoids unsupported assumptions

### Operational Readiness

- defines ownership
- includes escalation
- supports repeatable execution
- includes troubleshooting

### Governance Quality

- defines controls
- captures evidence
- includes review cadence
- supports auditability

### Portfolio Quality

- communicates clearly
- uses reusable artifacts
- demonstrates implementation skill
- supports architecture discussion
