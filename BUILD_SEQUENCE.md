# Academy Release Sequence

## Purpose

The release sequence defines how the UKG WFM Academy is organized into coherent public curriculum releases. Each release creates a complete learning layer that can stand alone while preparing learners for deeper domain, technical, governance, and architecture work.

The sequence is structured around enterprise usefulness:

- establish shared standards
- build foundation knowledge
- deepen core WFM operations
- add technical engineering capability
- add analytics and automation systems
- create governance and implementation playbooks
- package capstones and certifications

## Release 0: Academy Governance Foundation

Release 0 establishes the academy operating model.

## Release Quality Gates

Every repository release must include:

- complete repository structure
- module map

- lab package
- architecture documentation
- governance controls
- troubleshooting guidance
- validation checklist
- assessment rubric
- capstone simulation
- portfolio deliverable

Every release must support:

- independent learner use
- instructor-led delivery
- GitHub portfolio review
- enterprise implementation discussion
- architecture review
- governance review
- certification assessment
